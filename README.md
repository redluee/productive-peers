# Productive Peers

Productive Peers is a Flutter-based mobile application designed to help users track and manage their personal goals, habits, and studies in a structured and motivating way.

## Project Goal

The primary goal of this project is to create a beautiful and intuitive application that empowers users to stay productive and achieve their objectives. It aims to provide different tools for different types of goals, recognizing that a "one-size-fits-all" approach is not effective for personal development.

## Features

The application will distinguish between three main types of goals:

### 1. Habits
For recurring activities, the app will focus on building consistency.
- **Streak Tracking**: To motivate users, the app will display a streak (like on Snapchat) for consecutive days a habit is completed on its scheduled day.
- **Daily Check-ins**: A simple way to mark a habit as "done" for the day.

### 2. Goals
For larger objectives, the focus is on incremental progress.
- **Progress Bar**: A clear visual indicator of how far the user has come.
- **Milestones**: Users can break down their goals into smaller, manageable milestones, each tied to a specific percentage of completion.

### 3. Studies
For learning-based objectives, the application will help with time management.
- **Date-based Milestones**: Milestones are associated with specific deadlines to keep the user on track.
- **Timeline View**: A visual representation of the study schedule, showing upcoming and overdue milestones.
